Using FamilySearch to Find Missing Persons

If you're looking for genealogical information, this website can help you trace family history and connections.

  1. Access the FamilySearch website: Visit the FamilySearch website at https://www.familysearch.org.
  2. If you have any genealogical information about the missing person: Enter it into the search fields provided.
  3. Browse the search results: Identify potential family connections or historical records related to the individual by reviewing the search results.
